The BASE OAI-PMH Validator is a service provided by Bielefeld Academic Search Engine (BASE). It has been designed to help repository managers to easily verify if their repository is compliant with the BASE requirements.

To validate your repository, enter its OAI-PMH endpoint URL and press "Validate!".

Note: We use caching to minimize the number of requests to the repository server. However, depending on the responsiveness of the server, the verification process may take some time. Also, each request currently times out after 20 seconds.
